# ESPBrew WASM UI

This directory contains the WebAssembly UI for ESPBrew.

## Files

- `index.html` - WASM loader and application entry point
- `main.wasm` - Compiled Go WASM module (generated by `go run cmd/wasm-compiler`)
- `wasm_exec.js` - Go WASM runtime from Go SDK
- `styles.css` - Application styles

## Building WASM

From the project root:

```bash
# Using the helper command
go run cmd/wasm-compiler

# Or manually
GOOS=js GOARCH=wasm go build -o web/main.wasm ./internal/ui
```

## Copying wasm_exec.js

If `wasm_exec.js` is missing, copy it from your Go SDK:

```bash
cp $(go env GOROOT)/misc/wasm/wasm_exec.js web/
```

## Accessing the WASM UI

- `/` - Legacy HTML interface
- `/v2/` - New WASM interface (this directory)

## Development

The WASM UI is built with pure Go:

- `internal/ui/dom` - DOM manipulation helpers
- `internal/ui/components` - Reusable UI components
- `internal/ui/layout` - Layout components (navbar, tabs, etc.)
- `internal/ui/pages` - Page implementations
- `internal/ui/api` - API client

No npm, no JavaScript frameworks - 100% Go.
